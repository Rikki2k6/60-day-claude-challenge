# Day 20 - Face Puzzle Game

## Objective

The objective of Day 20 was to build an interactive **Face Puzzle Game** using Claude. The application captures a photo using the webcam, converts it into a sliding puzzle, and allows users to solve it while tracking their performance through timers, move counters, and a local leaderboard.

---

## Task Overview

During this challenge, I:

- Generated the complete HTML Face Puzzle Game using Claude.
- Saved and opened the HTML application in the browser.
- Allowed webcam permissions and captured a photo.
- Generated a custom face puzzle.
- Selected the **5×5** difficulty level.
- Played the puzzle using drag-and-drop interactions.
- Verified timer, move counter, and leaderboard functionality.
- Successfully completed the puzzle.
- Saved screenshots and the generated HTML application.

---

## Features

- 📷 Webcam photo capture
- 🧩 Dynamic puzzle generation
- 🎮 Multiple difficulty levels (3×3, 4×4, 5×5)
- ⏱️ Real-time timer
- 🔢 Move counter
- 🏆 Local leaderboard with best completion times
- 🖱️ Drag-and-drop gameplay
- 📱 Touch-friendly interactions
- 🎉 Puzzle completion summary
