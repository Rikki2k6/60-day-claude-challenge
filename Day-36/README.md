# Cognitive Pattern Explorer

Today I built an interactive **Cognitive Pattern Explorer** using Claude. The application guides users through a series of reflective activities to explore their natural thinking tendencies. Instead of measuring intelligence or personality, it focuses on recognizing decision-making patterns through interactive scenarios, prioritization exercises, and thinking-sequence mapping.

The experience concludes with a personalized Reflection Journal summarizing the user's dominant cognitive style.

---

## Interactive Experience

The application consists of four stages:

### Calm / Stress Mode
- Choose between Calm Mode and Stress Mode.
- Theme, animation speed, and overall atmosphere adapt based on the selected mode.

### Chapter 1 – Everyday Scenarios
- Respond to multiple real-life situations.
- Each choice contributes toward different cognitive tendencies.

### Chapter 2 – Priority Sorting
- Drag and drop priority cards into the order that best reflects personal decision-making.
- Supports both mouse and keyboard interactions.

### Chapter 3 – Thinking Sequence
- Arrange the natural sequence of thoughts during important decisions.
- Uses an interactive timeline with drag-and-drop functionality.

### Reflection Journal
At the end of the experience, the application generates:

- Primary thinking pattern
- Percentage distribution across thinking styles
- Visual cognitive profile
- Personalized reflection journal
- Educational disclaimer
- Option to replay with another mode

## What I Learned

### 1. Human thinking isn't fixed

Decision-making changes depending on context, emotions, and circumstances rather than following one rigid pattern.

### 2. Reflection creates awareness

Simple interactive exercises can reveal habits that usually go unnoticed during everyday decisions.

### 3. UI can improve engagement

Turning psychological reflection into an interactive experience makes the learning process more engaging and memorable.

### 4. Visualization improves understanding

Displaying thinking styles with percentage bars and a visual profile makes complex information easier to interpret.

### 5. AI can build educational simulations

Large Language Models can generate complete interactive web applications combining HTML, CSS, and JavaScript without external frameworks.

## Conclusion

This project demonstrated how web technologies and AI can be combined to create engaging self-reflection experiences. Instead of providing a psychological assessment, the application encourages users to observe their own thinking habits through interactive activities.

The biggest takeaway from this task is that understanding **how** we think can be just as valuable as knowing **what** we think.
