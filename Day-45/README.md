# Day 45 – AI Decision Report Dashboard

## 📌 Overview

For **Day 45** of the **ABTalks 60 Days Challenge**, I built an **AI-powered Decision Report Dashboard** using Claude. The application conducts a structured interview, evaluates multiple options using a decision matrix, identifies hidden assumptions and cognitive biases, and generates an interactive dashboard with actionable insights.

The focus of this project was to explore how AI can assist in structured decision-making rather than simply providing direct answers.

---

## ✨ Features

- 🧠 Guided multi-step decision interview
- 📊 Interactive decision matrix with animated score bars
- ⚖️ Side-by-side option comparison
- 🔍 Assumption & cognitive bias analysis
- ⚠️ Premortem risk assessment
- 📅 7-Day validation plan
- 🏆 AI-generated recommendation
- 📸 Shareable summary cards
- 🎨 Modern responsive UI with smooth animations
